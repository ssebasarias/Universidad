# Daniel Alejandro Lopez  
## Taller Minería de Datos  

### Objetivo  
El estudiante aprenderá sobre análisis de datos en bases de datos no relacionales.  

### Herramientas  
- **MongoDB**  

### Datos  
[European Soccer Database - Kaggle](https://www.kaggle.com/datasets/abdelrhmanragab/european-soccer-database?select=Player.csv)  

### Paso a Paso  

1. **Organización de Tablas**  
    - En Excel, organiza las tablas y encuentra las relaciones entre ellas.  

2. **Modelo Entidad-Relación (MER)**  
    - Crea un MER en una base de datos SQL Server o MySQL.  
    - Define las tablas y las relaciones entre ellas.  

3. **Consulta y Exportación**  
    - Crea una consulta SQL que una todas las tablas mediante `JOINS`.  
    - Exporta manualmente los datos obtenidos a un archivo Excel.  

4. **Conversión a CSV**  
    - Convierte el archivo Excel a formato CSV.  

5. **Instalación de MongoDB**  
    - Instala MongoDB en tu sistema.  

6. **Importación de Datos**  
    - Importa el archivo CSV a MongoDB.  

7. **Consultas en MongoDB**  
    - Realiza una consulta que traiga todos los goles de Lionel Messi.  

8. **Actualización de Datos**  
    - Actualiza en MongoDB el nombre de la liga inglesa a "Premier League".  

9. **Comparación de Goles**  
    - Realiza una consulta que compare los goles de Cristiano Ronaldo vs Lionel Messi.  

10. **Eliminación de Datos**  
     - Elimina el último partido del equipo Liverpool de la última jornada (`state`).  

11. **Análisis de Goles por Liga**  
     - Realiza una consulta que compare los goles de las diferentes ligas en todas las temporadas.  

### Nota  
Envía pantallazos en un archivo Word del paso a paso realizado.
